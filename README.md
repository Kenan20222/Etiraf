## Linux'da Qurulum:

- 1. Node JS ve Telegraf'ı qurun.
```
sudo apt install nodejs
sudo apt install npm
npm install telegraf --save
```
- 2. Repo'yu Klonlayın.
```
git clone https://github.com/sirincay/EtirafBot
cd EtirafBot
```
- 3. Configdə Tokeni Ayarlayın.
```
nano config.js
```
- 4. Botu Başladın.
```
node bot.js
```
